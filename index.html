Features

Visual Cleanup Only

ブラウザ上の表示（CSS/DOM）のみを調整します。APIや通信には一切干渉せず、安全に利用できます。

Privacy First

ユーザーデータは一切収集しません。すべての処理はあなたのブラウザ内で完結します。

Lightweight

余計な負荷をかけない軽量設計。Xの動作速度に影響を与えません。

Privacy Policy

Effective Date: 2026-02-16

1. Data Collection

This extension ("premium cleaner for X") does not collect, store, or transmit any user data. It operates entirely locally within your browser.

2. Permissions

The extension requires permission to access data on x.com and twitter.com solely for the purpose of modifying the visual appearance of the web page (hiding specific elements). No data is read for any other purpose.

3. Third-Party Services

This extension does not use any third-party analytics or tracking services.

4. Changes to This Policy

We may update our Privacy Policy from time to time. Thus, you are advised to review this page periodically for any changes.
